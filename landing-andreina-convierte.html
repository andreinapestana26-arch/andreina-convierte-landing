Preguntas frecuentes

Todo lo que necesitas saber.

¿Para quién es este programa? +
Para dueños de PyMEs, gerentes comerciales y equipos de ventas en Latinoamérica que venden bien algunos días y mal otros — porque no tienen un proceso definido. Si tú o tu equipo improvisan cada venta, este programa es para ti.
¿Necesito experiencia previa en ventas? +
No. El programa empieza desde el diagnóstico — funciona tanto si eres nuevo en ventas como si llevas años en esto y sientes que hay un techo que no puedes romper. El módulo 1 te ubica exactamente donde estás.
¿Cómo funcionan las herramientas con IA? +
Cada herramienta está entrenada con la metodología, voz y filosofía de Andreina. Tú describes tu situación, y la IA te devuelve un análisis, plan o simulación personalizada — como si le preguntaras directamente a Andreina. Funcionan en el navegador, sin instalaciones.
¿Qué incluye la suscripción de $9/mes? +
Las 4 herramientas bonus activas: Tracker de Prospectos, Tracker de Productos, Generador de Scripts para Servicios y Generador de Scripts para Productos. También incluye actualizaciones futuras de todas las herramientas. Puedes cancelar cuando quieras sin penalidades.
¿El programa sirve para equipos o solo para uno? +
Las metodologías y herramientas están diseñadas para que las uses en tu negocio con tu equipo. El módulo de diagnóstico puede aplicarse a cualquier integrante del equipo. Para uso simultáneo de múltiples personas, contáctame directamente para opciones de equipo.
¿Puedo acceder desde cualquier país de LATAM? +
Sí. El programa es 100% digital y funciona desde cualquier país. Los precios están en USD para facilitar el acceso regional. El pago se procesa de forma segura a través de Hotmart, que acepta múltiples métodos de pago según tu país.
